## 场景划分

| 场景 ID | 名称 | 尺寸(格) | 描述 |
|---------|------|---------|------|
| `village` | 福音小镇（主地图） | 120×80 | 户外主场景：广场、街道、建筑外观、公园 |
| `tavern` | 星落酒馆 | 20×16 | 室内：吧台、座位、壁炉（赵大厨经营，全时段餐饮） |
| `shop` | 杂货铺 | 16×12 | 室内：柜台、货架 |
| `clinic` | 镇医院 | 16×12 | 室内：诊台、药柜（苏医生坐诊） |
| `school` | 村学堂 | 20×16 | 室内：黑板、课桌 |
| `home_li` | 李家 | 16×12 | 室内：床、桌椅 |
| `home_wang` | 王家 | 16×12 | 室内 |
| `home_zhao` | 赵家 | 16×12 | 室内 |

## 地标与兴趣点

| 地标 | 位置(村庄) | 类型 | 说明 |
|------|-----------|------|------|
| 村广场 | (55-65, 35-45) | 圆形石板区 | 村中心，有喷泉，居民聚集地 |
| 主街道 | y=38~42, x=10~110 | 石板路 | 横贯东西的主干道 |
| 纵向街道 | x=58~62, y=10~70 | 石板路 | 南北纵贯 |
| 村公园 | (40-80, 55-70) | 草地+树木 | 长椅、小池塘、樱花树 |
| 镇医院 | (50-70, 12-22) | 大型建筑 | 苏医生坐诊，提供医疗和心理咨询 |
| 布告栏 | (58, 34) | 1×1 物件 | 村公告/任务板 |
| 水井 | (45, 40) | 1×1 物件 | 装饰+互动点 |
