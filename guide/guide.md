# 🏘️ 福音小镇 (Gospel Town) — 项目总览

> **灵感来源**: 星露谷物语 · 鹬鹕镇  
**技术路线**: 纯前端架构（HTML5 Canvas + Vanilla JS + LLM API）  
**LLM 支持**: GLM-4-Flash（云端）/ Qwen3-14B-Q8 via Ollama（本地，推荐）  
**核心目标**: 8 位 AI 居民的像素风村落，居民能自主生活、社交、工作，玩家可以观察/对话
**当前版本**: v2.0
